# Marsmini-
https://npas-78659.web.app/

User는 mars mini게임을 하면서 얻은 포인트를 관리하고, 관리자는 user를 관리하기 위한 사이트이다.

현재 보유한 포인트를 통해 코인으로 출금을 신청하고, 그동안의 출금 내역과 포인트 내역, 초대내역을 확인할 수 있다.

관리자의경우 사용자의 출금신청을 수락 또는 거절하고 사용자의 상세정보를 확인 할 수 있다.
